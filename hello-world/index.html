Hello World

Our first program is a very simple example, but it's the bare minimum to getting you started in Node.js. It will print the classic "Hello World" message.

'use strict';

console.log('Hello World');

console is a global variable available in all modules.

To run the program, put the code in hello-world.js and run

node hello-world.js
